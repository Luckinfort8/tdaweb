Etiquetas HTML

'dl'

Representa una lista descriptiva. El elemento encierra una lista de grupos de términos.


'dt'

Especifica un término en una descripción o lista de definiciones, y como tal debe utilizarse dentro de un elemento 'dl'.


'dd'

Provee detalles acerca de o la definición de un término precedente 'dt' en una lista de descripciones 'dl'


'blockquote'

blockquote -cita en bloque . Crea citas en bloque, marca las citas a otros autores o documentos.


'q'

Indica que el texto adjunto es una cita corta en línea. La mayoría de los navegadores modernos implementan esto rodeando el texto entre comillas.


'cite'

Marca una referencia a una fuente, o el autor de un texto citado.


'abbr'

Representa una abreviación o acrónimo; el atributo opcional title puede ampliar o describir la abreviatura.


'address'

Aporta información de contacto para su 'article' más cercano o ancestro 'body'; en el último caso lo aplica a todo el documento.


'sub'

Define un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmente más pequeño, que el tramo principal del texto, es decir, en subíndice.


'sup'

Define un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño, que el tramo principal del texto, es decir, en superíndice.


'code'

Es el apropiado para marcar el código de un programa.


'pre'

Representa texto preformateado. El texto en este elemento típicamente se muestra en una fuente fija, no proporcional, exactamente como es mostrado en el archivo.


'var'

Marca variables de programas y similares.

el bucle no se detendrá hasta que Faltan sea igual a 0.


'samp'

El elemento HTML Sample se utiliza para incluir texto en línea que representa una muestra (o cita) de la salida de un programa de ordenador.


'time'

Representa un periodo específico en el tiempo. Puede incluir el atributo datetime para convertir las fechas en un formato interno legible por un ordenador, permitiendo mejores resultados en los motores de búsqueda o características personalizadas como recordatorios.
